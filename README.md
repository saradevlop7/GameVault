# 🎮 GameVault App

## 📌 Description
GameVault est une application web permettant de consulter un catalogue de jeux vidéo et de gérer un panier d’achat de manière dynamique.

## 🚀 Fonctionnalités
- Affichage des jeux sous forme de cartes  
- Recherche par titre  
- Filtrage par catégorie  
- Ajout au panier  
- Gestion des quantités  
- Suppression des produits  
- Sauvegarde avec LocalStorage  
- Design responsive avec Bootstrap  

## 🛠️ Technologies
- HTML  
- JavaScript  
- Bootstrap
